## Featured Projects

### ERP Agentic RAG System
> Multi-tenant, role-aware Q&A over live ERP data

Natural-language queries are classified and routed to a RAG pipeline (document search), a SQL pipeline (live PostgreSQL queries), or both in Hybrid mode — behind a 5-layer middleware stack (auth, RBAC, rate limiting, logging, metrics).

**Stack:** Python · FastAPI · PostgreSQL · Gemini · JWT RS256 · Prometheus · Docker

---

### LoopStack — Automation Hub
> AI agency delivering intelligent automation for enterprises

Combines n8n workflow orchestration with custom AI agents to automate business operations end-to-end. Includes an execution auditor that analyzes workflow runs, surfaces bottlenecks, estimates AI token costs, and generates PDF reports.

**Stack:** n8n · Python · React · TypeScript · Docker

---

### SmartCampus Companion
> Cross-platform mobile app for university campus life

Real-time campus services, scheduling, and resource management in a unified Flutter application with backend synchronization.

**Stack:** Flutter · Firebase · FastAPI

---

### ChronicCare — Health-Tech Hackathon
> AI-assisted chronic disease management platform

Built at a national hackathon. Backend with patient data pipelines, AI nurse assistant, and Google Calendar/OAuth integration for appointment flows.

**Stack:** Python · FastAPI · Firebase · PostgreSQL · Google APIs
